【城市藝站 |徵稿啟事】

大家好，城市藝站的第一個活動登場囉~ 歡迎你與我們一同參加這次的徵稿活動，以下是詳細的活動說明!

1.活動主旨 :
「喀嚓!」，為何不和我們一起拿起相機、按下快門，與公共藝術留下一張美好的回憶呢?
不知道你是否有注意過，公共藝術恰巧是許多人會停下來拍照打卡的據點之一，然而想必大部分的人應該都已經厭煩於一般的拍照手勢了吧，因此這個活動中融入了創意性及趣味性，考驗大家如何運用自己與公共藝術的互動來創造出獨一無二的合照，不僅可以讓公共藝術變得更生動活潑，也有助於讓人們更容易地了解處於我們生活之中的公共藝術。
歡迎你來探訪這些公共藝術，將你的創意及想像留存在這張獨特的合照之中吧!

2.徵稿主題 :
強調透過與公共藝術互動之徵稿活動

3.徵稿內容 :
從城市藝站所規劃之三條公共藝術路線中包含之公共藝術(共30個)擇一，拍攝創意互動照，詳情請見附件。

4.活動日期:
2019年9月4日(三)-2019年10月31日(四)23:59截止(GMT)
2019年11月5日(二)-2019年11月12日(二)進行線上投票
預定2019年11月15日(五)發表結果

5.參賽資格:
不限國籍與年齡，任何擁有創意的人士皆可參加。

6.參賽細則:
-作品規範
參賽作品須為數位檔案，每個人最多可寄出三張作品，不限彩色或黑白。
每件作品尺寸限定為1024*768，上傳不超過1MB為限，為JPEG或JPG格式。
作品寄出後會於24小時內回覆。
參賽作品需符合攝影主題及收件規格，作品可調整亮度、對比度、色彩飽和度、銳利度、裁切及運用電腦進行數位後製。
繳交作品若未能提供或提供檔案不符規格，將視同放棄參賽資格，主辦單位有權取消其參賽資格。
獲獎作品的著作權歸創作者自身所有，但城市藝站會以宣傳為目的使用在本團隊所經營的網站、服務。
不符合報名條件的作品、不符合主題的作品、城市藝站判斷為不適當的作品，將不會公開至相簿，且不另行通知參賽者未公開之理由。
-參賽流程
請依作品規範繳交作品至 ( 109publicartstation@gmail.com )，報名時須提供參賽者的作品名稱，電子郵件地址，作品理念等 ( 詳情請見附件 )。如獲獎通知等，在必要時會透過「109publicartstation@gmail.com」的電子郵件向您聯繫。
參賽者需於10/31日比賽截止前按讚此參賽貼文並tag兩位好友(即有機會抽中城市藝站周邊)。
若完成以上步驟即完成報名程序，有任何疑問，歡迎來信詢問，或上FB搜尋「城市藝站」粉絲專頁詢問小編!

7.評選與獎項:
-評選
公布作品以提供民眾線上票選，以得票數評選名次，惟每位民眾僅得獲獎乙次，若民眾有重複獲獎，則重複獎項將依得票順位替補。( 如 : A參賽兩張作品，並分別獲得第二、三名，則第三名將由第四名之作品進行替補。)
-評分標準
各作品所獲得的票數會作為評審的參考﹐不過並不會僅依據票數來決定是否獲獎。
-評分比例
線上票選佔比50%，評審評分50%，由總分依序選出得獎者，取總分最高之前3名序位，前3高分者依序為特優、優等、佳作。
-獎項
特優 ( 一名 ) : 新光三越千元禮券一張
優等 ( 一名 ) : 電影票券兩張
佳作 ( 一名 ) : 博客來兩百元禮物卡一張
